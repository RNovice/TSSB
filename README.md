## Local usage

> may use python3 & pip3 in different os

optional venv：

```
python -m venv env
# Linux, macOS
env\bin\activate
# Windows
env\Scripts\activate
```

setup `.env` file：

```
DISCORD_WEBHOOK={{your discord webhooks url}}
```

run scripts：

```
pip install -r requirements.txt
python main.py
```
